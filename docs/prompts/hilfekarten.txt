Eigenschaften
{# Hilfearten - mehrere können gleichzeitig ausgewählt sein #}
{% if r|param:"Baustein_Hilfekarten: Hilfeart-Inhaltliche Hilfe" == "Ja" %}
Die Hilfekarte soll inhaltliche Hilfen enthalten. Der Fokus liegt darauf, wichtige Fachbegriffe, zentrale Konzepte und relevante Formeln verständlich zu erklären. Vermeide dabei komplexe Fachsprache und konzentriere dich auf das Wesentliche.
{% endif %}

{% if r|param:"Baustein_Hilfekarten: Hilfeart-Methodische Hilfe" == "Ja" %}
Die Hilfekarte soll methodische Hilfen bereitstellen. Sie soll den Lösungsweg strukturieren und konkrete Arbeitsschritte aufzeigen. Auch Tipps zur Recherche und Hinweise, worauf man achten muss, können, wenn sinnvoll, eingefügt werden. Die Lehrkraft hat die methodische Hilfe wie folgt beschrieben: "{{ r|param:"Baustein_Hilfekarten: Hilfeart-Methodische Hilfe - Beschreibung" }}"
Stelle die Vorgehensweise klar und nachvollziehbar dar.
{% endif %}

{% if r|param:"Baustein_Hilfekarten: Hilfeart-Sprachliche Hilfe" == "Ja" %}
Die Hilfekarte soll sprachliche Hilfen anbieten. Stelle wichtige Redemittel, Satzmuster und fachspezifische Formulierungen zur Verfügung. Biete Beispielformulierungen an, die als Orientierung dienen können.
{% endif %}

{# Hilfegrad - nur ein Grad kann ausgewählt sein #}
{% if r|param:"Baustein_Hilfekarten: Hilfegrad" == "Minimal" %}
Gestalte die Hilfen als subtile Impulse. Gib nur kleine Denkanstöße und Hinweise. Die Lernenden sollen weiterhin eigenständig denken müssen.
{% elif r|param:"Baustein_Hilfekarten: Hilfegrad" == "Mittlere" %}
Biete moderate Unterstützung an. Zeige wichtige Zwischenschritte auf und gib gezielte Hilfestellungen, die den Lösungsweg strukturieren. Lasse dabei noch Raum für eigenständiges Denken.
{% elif r|param:"Baustein_Hilfekarten: Hilfegrad" == "Umfangreiche" %}
Stelle detaillierte und umfassende Hilfen bereit. Erkläre jeden Schritt ausführlich und biete konkrete Beispiele an. Die Unterstützung soll so gestaltet sein, dass auch schwächere Lernende damit arbeiten können.
{% endif %}

{# Strukturierung - nur eine Strukturierungsart kann ausgewählt sein #}
{% if r|param:"Baustein_Hilfekarten: Strukturierung" == "Checkliste" %}
Strukturiere die Hilfen als übersichtliche Checkliste. Formuliere kurze, prägnante Checkpunkte, die nacheinander abgearbeitet werden können. Verwende aktive Verben zu Beginn jedes Punktes.
{% elif r|param:"Baustein_Hilfekarten: Strukturierung" == "Fließtext" %}
Präsentiere die Hilfen in Form eines zusammenhängenden, gut gegliederten Texts. Achte auf klare Absätze und einen logischen Aufbau. Verwende Verbindungswörter, um die einzelnen Aspekte zu verknüpfen.
{% elif r|param:"Baustein_Hilfekarten: Strukturierung" == "Stichpunkte" %}
Gliedere die Hilfen in prägnante Stichpunkte. Jeder Punkt soll eine zentrale Information oder einen wichtigen Aspekt enthalten. Achte auf eine klare Hierarchie und logische Reihenfolge der Punkte.
Achte darauf, keine ganzen Sätze zu formulieren, sondern wirklich nur Stichpunkte, also kurze und prägnante Aufzählungen, zu verwenden.
{% elif r|param:"Baustein_Hilfekarten: Strukturierung" == "Wenn-Dann-Schema" %}
Strukturiere die Hilfen nach dem Wenn-Dann-Prinzip. Stelle klare Bedingungen ("Wenn...") und die daraus folgenden Handlungsschritte ("Dann...") dar. Diese Form eignet sich besonders für regelbasierte oder algorithmische Aufgaben.
{% endif %}
