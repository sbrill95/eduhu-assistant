# Lernsituation-Prompt (Pflege) — aus alter Software

## Template-Variablen
- `LNG-BS-Beruf`: Ausbildungsberuf (z.B. Pflegefachmann/Pflegefachfrau)
- `LNG-BS-Jahr`: Ausbildungsjahr (1/2/3)
- `LNG-BS-LF`: Lernfeld ja/nein
- `LNG-BS-Lernfeld`: Konkretes Lernfeld/CE
- `LNG-BS-Fach`: Fach (wenn kein Lernfeld)
- `BS_Lernsituation_Inhalte`: Inhalte der Lernsituation

## Struktur
1. Rolle: Lehrkraft Berufsbildung, Beruf + Jahr
2. Kontext: Didaktische Grundsätze (Handlungsorientierung, Kompetenzorientierung, Situationsprinzip)
3. Unterrichtssituation: Lernfeld + Inhalte ODER Fach mit Berufsbezug
4. Jahrgangsdifferenzierung: 1.Jahr=Grundlagen, 2.Jahr=Komplex, 3.Jahr=Eigenverantwortlich
5. Curriculum-Inhalte pro CE (CE 01-04+ mit Bildungszielen, Kompetenzen, Lernanregungen)

## Erfasste CEs
- CE 01: Ausbildungsstart
- CE 02: Bewegung und Selbstversorgung (02A Mobilität, 02B Selbstversorgung)
- CE 03: Erste Pflegeerfahrungen reflektieren
- CE 04: Gesundheit fördern (allgemein + Kinder/Jugend + alte Menschen)

## Didaktische Grundsätze
1. Handlungsorientierung: praxisnah, berufliche Realität, aktives Handeln
2. Kompetenzorientierung: Fach-, Methoden-, Sozial-, Selbstkompetenz → berufliche Handlungskompetenz
3. Wissenschaftsprinzip + Situationsprinzip: fachlich korrekt + Arbeitsweltbezug

## Jahrgangsdifferenzierung
- Jahr 1: Grundlegende Konzepte, klare Strukturen, angeleitete Lernprozesse, überschaubare Fälle
- Jahr 2: Komplexere Zusammenhänge, zunehmend Verantwortung, mehrschichtige Situationen
- Jahr 3: Eigenverantwortliches Handeln, multiperspektivische Fälle, begründete Entscheidungen
